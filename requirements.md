# 5レンジャー ホームページ要件定義書

## 1. プロジェクト概要

### 1.1 プロジェクト名
株式会社Re.haru 5レンジャーサービス ホームページ

### 1.2 背景と目的
- 株式会社Re.haruの「5レンジャー」サービスを紹介するコーポレートサイト
- 「Return青春」という企業理念のもと、日本初の安心百均価格設計による一律料金システム開発を提供
- 一律500万円のシンプル料金、2〜6ヶ月の開発期間で、企業のDX推進を支援

### 1.3 目標
- サービス認知度の向上
- お問い合わせ・見積もり依頼の獲得
- 企業ブランディングの確立

## 2. 技術スタック ✅

- **フレームワーク**: Next.js 15 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **アニメーション**: Framer Motion
- **バックエンド**: Supabase
- **メール送信**: EmailJS
- **ホスティング**: Vercel

## 4. ブランディング要素 ✅

### 4.1 サービス名称
- **正式名称**: 5レンジャー（ゴレンジャー）
- **旧名称**: AIレンジャー → 2025年1月に5レンジャーへリブランド完了 ✅

### 4.2 価格表現
- **絶対値表現を採用** ✅:
  - 開発費用: 500万円（一律料金）
  - 開発期間: 2〜6ヶ月
  - 要件定義: 対話のみ
- **旧比較表現を完全廃止** ✅:
  - ~~1/5の価格~~ → 500万円
  - ~~1/3の期間~~ → 2〜6ヶ月
  - ~~1/10の工数~~ → 対話のみ

- **Return青春**: 誰もが青春のような「挑戦」「成長」「希望」を持ち続けられる社会を創る
